# Baseball Projects - Brady Nolin

A collection of data analysis and prediction models for Major League Baseball. These projects explore statistical modeling, machine learning, and sports analytics to predict playoff outcomes and analyze team performance metrics.

## Overview

This repository contains personal projects that combine my passion for baseball with data science. The work focuses on building predictive models and analytical tools using publicly available MLB data and advanced statistical techniques.

## Projects

### 1. **2025 Playoff Model** (`Prediction Models/2025 Playoff Model/`)
A sophisticated playoff prediction system using machine learning to forecast World Series odds and playoff series outcomes.

## Project Goals

- Build accurate predictive models for MLB outcomes
- Demonstrate proficiency in data analysis and machine learning
- Apply statistical concepts to real-world sports data
- Explore feature engineering and model validation techniques
